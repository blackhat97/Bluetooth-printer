# 블루투스 테스트 앱

Bluetooth Serial [Bluetooth Serial](https://ionicframework.com/docs/native/bluetooth-serial/)



## 설치 및 사용:

1. Install ionic 4.x
1. Fork this repo
1. execute any of following commands
```bash
$ npm install
$ ionic cordova run android -l --device
```
